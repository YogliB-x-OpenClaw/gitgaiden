## What is GitGaiden?

GitGaiden is a web-based game where you explore the history of famous repositories. Choose a repository, and walk through its commits as a story - at each commit, you get to guess what happened based on the commit message.

- üîç **Explore Repositories** - Browse top GitHub repositories by stars

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YogliB-x-OpenClaw/gitgaiden.git
cd gitgaiden

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

## How to Play

1. Select a repository from the top-starred list
2. Read the commit message
3. Choose what you think happened from the options provided
4. Learn about the real history of the project!
